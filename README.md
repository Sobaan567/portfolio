# Syed Sobaan Najmi Portfolio

Personal portfolio for Syed Sobaan Najmi, a Computer Engineer and aspiring AI/ML Engineer. Built with React, Vite, and custom CSS animations.

## Tech Stack

- React
- Vite
- Lucide React
- CSS animations
- Static assets in `public/`

## Local Development

```bash
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## Production Build

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.

## Deploy To Vercel

1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Use these settings:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Install Command: `npm install`
4. Deploy.

## Project Structure

```text
public/        Static images and project screenshots
src/main.jsx   React app and portfolio content
src/styles.css Global styling and animations
```
